# NIFTY100 Financial Intelligence Platform

An end-to-end financial analytics platform for NIFTY 100 companies built using Django, PostgreSQL, React, Pandas, and Machine Learning.

## Features

* Financial data warehouse using PostgreSQL
* ETL pipeline for automated data ingestion
* Company financial analysis dashboard
* Health score and ranking system
* REST API using Django REST Framework
* Interactive React frontend
* Company comparison and sector analytics
* Data quality monitoring and validation
